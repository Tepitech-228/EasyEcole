Module E-learning — Précisions

1. Compression vidéo

Comment gérer la compression ?

A

Compression backend (FFmpeg)

À l'upload, l'API compresse la vidéo automatiquement. Pas de dépendance externe.

B

Service cloud (Cloudinary/Mux)

Upload vers un service externe qui gère compression et streaming.

C

Multi-qualité

Proposer 2 versions (haute/basse qualité) à l'upload, l'étudiant choisit.

2. Chat

Quel type de chat pour les dates d'évaluation ?

A

Chat simple (par cours)

Fil de discussion par cours, enseignants publient les dates, étudiants voient. Pas de temps réel.

B

Chat temps réel (Socket.io)

Messagerie instantanée par cours avec Socket.io. Plus complexe.

C

Système d'annonces

Les enseignants publient des annonces (dates éval, examens) sur le cours. Les étudiants reçoivent une notification. Pas de chat.
